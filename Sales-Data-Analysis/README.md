# Sales Data Analysis

## Project Overview
This project analyzes sales data to discover business insights using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Analysis Performed
1. Load dataset
2. Explore the data
3. Check data types and missing values
4. Calculate total sales
5. Find top-selling products
6. Analyze sales by category
7. Analyze sales by region
8. Create visualizations using Matplotlib

## Key Insights
- Total sales calculated
- Best-selling products identified
- Category-wise sales analyzed
- Regional sales performance visualized
